# CodePulse Privacy Policy

CodePulse does not collect, transmit, or store any personal data on external servers.

All activity data is:
- Generated locally
- Stored locally in the user's machine
- Pushed only to a Git repository explicitly configured by the user

CodePulse never:
- Sends data to third-party services
- Tracks usage analytics
- Accesses files outside the user's workspace activity

Git operations are performed only when Git sync is enabled and a repository URL is provided by the user.
